初始化和版本 (Initialization and Versions)

初始化和版本 (Initialization and Versions) — 库的版本号。

Synopsis

#include <gdk-pixbuf/gdk-pixbuf.h>

extern              const char *gdk_pixbuf_version;
extern              const guint gdk_pixbuf_major_version;
extern              const guint gdk_pixbuf_minor_version;
extern              const guint gdk_pixbuf_micro_version;
#define             GDK_PIXBUF_VERSION
#define             GDK_PIXBUF_MAJOR
#define             GDK_PIXBUF_MINOR
#define             GDK_PIXBUF_MICRO

Description

可以使用这些宏和变量来检测您所链接的 gdk-pixbuf 的版本号。

Details

gdk_pixbuf_version

extern const char *gdk_pixbuf_version;

包含 gdk-pixbuf 库的完整版本号的字符串。这个是运行的程序使用的当前版本号。

gdk_pixbuf_major_version

extern const guint gdk_pixbuf_major_version;

gdk-pixbuf 库的主版本号。 （例如 gdk-pixbuf 版本1.2.5 的主版本号为 1）

这是个库中的变量，所以表示的是您链接的 gdk-pixbuf 库的主版本号。对比 GDK_PIXBUF_MAJOR 宏，这个宏表示的是您的程序包含的 gdk-pixbuf 头文件的主版本号。

gdk_pixbuf_minor_version

extern const guint gdk_pixbuf_minor_version;

gdk-pixbuf 库的次版本号。 （例如 gdk-pixbuf 版本1.2.5 的次版本号为 2）

这是个库中的变量，所以表示的是您链接的 gdk-pixbuf 库的次版本号。对 比 GDK_PIXBUF_MINOR 宏，这个宏表示的是您的程序包含的 gdk-pixbuf 头文件的次版本号。

gdk_pixbuf_micro_version

extern const guint gdk_pixbuf_micro_version;

gdk-pixbuf 库的修订版本号。 （例如 gdk-pixbuf 版本1.2.5 的修订版本号为 5）

这是个库中的变量，所以表示的是您链接的 gdk-pixbuf 库的修订版本号。对 比 GDK_PIXBUF_MICRO 宏，这个宏表示的是您的程序包含的 gdk-pixbuf 头文件的修订版本号。

GDK_PIXBUF_VERSION

#define GDK_PIXBUF_VERSION "2.22.1"

包含 gdk-pixbuf 头文件的完整版本号的字符串。与 gdk_pixbuf_version 对比， 这个是正被编译的版本号。

GDK_PIXBUF_MAJOR

#define GDK_PIXBUF_MAJOR (2)

gdk-pixbuf 库的主版本号，比如版本"0.8.0"中的第一个"0"就是主版本号。

GDK_PIXBUF_MINOR

#define GDK_PIXBUF_MINOR (22)

gdk-pixbuf 库的次版本号，比如版本"0.8.0"中的"8"就是次版本号。

GDK_PIXBUF_MICRO

#define GDK_PIXBUF_MICRO (1)

gdk-pixbuf 库的修订版本号，比如版本"0.8.0"中的最后一个"0"就是修订版本号。
