I. API 参考
初始化和版本 (Initialization and Versions) — 库的版本号。
GdkPixbuf 结构(The GdkPixbuf Structure) — 描述一个图像的信息。
引用计数和内存管理 (Reference Counting and Memory Mangement) — 引用计数和内存管理所用到的函数。
加载文件(File Loading) — 从一个文件加载一个 pixbuf。
保存文件(File saving) — 保存一个 pixbuf 到文件。
内存中的图象数据(Image Data in Memory) — 使用内存中的数据创建一个 pixbuf 。
内联数据 (Inline data) — pixbuf 处理内联数据的函数。
缩放 (Scaling)
渲染 — 将一个 pixbuf 渲染到一个 GDK drawable。
Drawables to Pixbufs — 获取一个 GDK drawable 的一部分图像数据，并将其转换到 pixbuf。
工具 (Utilities) — 工具和混杂的便捷函数。
Animations — 动画。
GdkPixbuf加载器(GdkPixbufLoader) — 应用驱动先进图像加载。
模块接口 (Module Interface) — gdk-pixbuf 的扩展。
gdk-pixbuf Xlib 初始化 (gdk-pixbuf Xlib initialization) — gdk-pixbuf Xlib 库的初始化。
渲染 Xlib (Xlib Rendering) — 将一个 pixbuf 渲染到 X drawable。
从X Drawable 到 Pixbuf (X Drawables to Pixbufs) — 从一个 X drawable 取得部分图像数据并放入 pixbuf。
XlibRGB — 将 RGB 缓冲区渲染到 X drawable。
II. 工具(Tools)参考
gdk-pixbuf-csource — 为 GdkPixbuf 图像生成 C 代码的工具。
gdk-pixbuf-query-loaders — GdkPixbuf loader 注册工具
