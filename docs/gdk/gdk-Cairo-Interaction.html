Description

开罗(Cairo)是GDK使用的 支持矢量和点阵的图形库。从GTK+ 2.8版本之后，GTK+ 主要使用开罗(Cairo)进行绘画。

GDK没有封装开罗(Cairo) API, 但是允许创建开罗上下文来绘制界面。 另外，GDK函数允许转换GDK所构建的长方体区域(rectangles and regions)到开罗 (Cairo)的路径(paths)中, 以便使用开罗进行绘图和着色操作。

gdk_cairo_create ()

cairo_t *           gdk_cairo_create                    (GdkDrawable *drawable);

Creates a Cairo context for drawing to drawable.

Note

Note that due to double-buffering, Cairo contexts created in a GTK+ expose event handler cannot be cached and reused between different expose events.

 a GdkDrawable

 A newly created Cairo context. Free with cairo_destroy() when you are done drawing.
